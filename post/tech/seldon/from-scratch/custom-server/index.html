Pre-requisites

이번 포스트에서는 구성하는 Custom Server를 사용하기 위해서는 아래와 같은 Secret이 필요합니다.

  1. GHCR secrets
  2. aws secrets

1. GHCR Secrets

작성한 Custom Server의 Docker는 GHCR에서 관리합니다.
k8s에서 image를 pull 하기 위해서는 ghcr에 접근할 수 있는 권한이 필요합니다.

1kubectl -n seldon create secret docker-registry mrx-ghcr \\
2   --docker-server=ghcr.io \\
3   --docker-username=<github id> \\
4   --docker-password=<secret key> \\
5   --docker-email=<github mail>

2. aws secrets

이번 포스트에서는 모델을 S3에 저장한다고 가정합니다.
k8s에서 S3에 있는 모델을 가져오기 위한 aws에 접근할 수 있는 권한이 필요합니다.
이를 위해서 아래와 같은 aws-secret.yaml을 작성합니다.

 1apiVersion: v1
 2kind: Secret
 3metadata:
 4 name: seldon-init-container-secret
 5 namespace: seldon
 6type: Opaque
 7data:
 8 AWS_ACCESS_KEY_ID: <base64 | key_id>
 9 AWS_SECRET_ACCESS_KEY: <base64 | access_key>
10 AWS_ENDPOINT_URL: <base64 | endpoint>
11 USE_SSL: <base64 | true>

여기서 주의해야할 점은 data에 value 값은 모두 base64 인코딩 되어 있어야 합니다.
이는 bash에서 base64 command로 쉽게 변환할 수 있습니다.

1> echo 'hello, world!' | base64
2aGVsbG8sIHdvcmxkIQo=

apply로 secret을 생성합니다.

1> kubectl apply -f aws-secret.yaml
2secret/seldon-init-container-secret created

Custom Server

Seldon-core에서 제공하는 pre-packaged inference server 다음과 같습니다.

  • SKLearn Server
  • MLflow Server
  • Triton Inference Server
  • Tensorflow Serving
  • XGBoost Server

대부분의 경우는 제공하는 서버를 사용하면 됩니다. 하지만 필요한 경우 Custom 서버를 직접 만들어서 사용할 수 도 있습니다.
이번 포스트에서는 MLFlowServer를 기본으로 하지만 Conda 환경이 아닌 Pip 환경을 이용하는 방법에 대해서 설명합니다.

 29class PyFuncModel:
 30    """
 31    MLflow 'python function' model.
 32    Wrapper around model implementation and metadata. This class is not meant to be constructed
 33    directly. Instead, instances of this class are constructed and returned from
 34    :py:func:`load_model() <mlflow.pyfunc.load_model>`.
 35    ``model_impl`` can be any Python object that implements the `Pyfunc interface
 36    <https://mlflow.org/docs/latest/python_api/mlflow.pyfunc.html#pyfunc-inference-api>`_, and is
 37    returned by invoking the model's ``loader_module``.
 38    ``model_meta`` contains model metadata loaded from the MLmodel file.
 39    """
 40
 41    ## Customized from mlflow==1.16.0
 42    def __init__(self, model_meta: Model, model_impl: Any):
 43        if not model_meta:
 44            raise MlflowException("Model is missing metadata.")
 45        self._model_meta = model_meta
 46        self._model_impl = model_impl
 47
 48    def predict(self, data: PyFuncInput) -> PyFuncOutput:
 49        """
 50        Generate model predictions.
 51        If the model contains signature, enforce the input schema first before calling the model
 52        implementation with the sanitized input. If the pyfunc model does not include model schema,
 53        the input is passed to the model implementation as is. See `Model Signature Enforcement
 54        <https://www.mlflow.org/docs/latest/models.html#signature-enforcement>`_ for more details."
 55        :param data: Model input as one of pandas.DataFrame, numpy.ndarray, or
 56                     Dict[str, numpy.ndarray]
 57        :return: Model predictions as one of pandas.DataFrame, pandas.Series, numpy.ndarray or list.
 58        """
 59        input_schema = self.metadata.get_input_schema()
 60        if input_schema is not None:
 61            data = _enforce_schema(data, input_schema)
 62        return self._model_impl.predict(data)
 63
 64    ## Customized from mlflow==1.16.0
 65    def transform(self, data: PyFuncInput) -> PyFuncOutput:
 66        """transform method"""
 67        input_schema = self.metadata.get_input_schema()
 68        if input_schema is not None:
 69            data = _enforce_schema(data, input_schema)
 70        return self._model_impl.transform(data)
 71
 72    @property
 73    def metadata(self) -> Model:
 74        """Model metadata."""
 75        if self._model_meta is None:
 76            raise MlflowException("Model is missing metadata.")
 77        return self._model_meta
 78
 79    def __repr__(self) -> Any:
 80        info = {}
 81        if self._model_meta is not None:
 82            if (
 83                hasattr(self._model_meta, "run_id")

 90            ):
 91                info["artifact_path"] = self._model_meta.artifact_path

 93            info["flavor"] = self._model_meta.flavors[FLAVOR_NAME]["loader_module"]

 96            {"mlflow.pyfunc.loaded_model": info},
 97            default_flow_style=False,

100def load_model(model_uri: str, suppress_warnings: bool = True) -> PyFuncModel:
101    local_path = _download_artifact_from_uri(artifact_uri=model_uri)
102    model_meta = Model.load(os.path.join(local_path, MLMODEL_FILE_NAME))
103
104    conf = model_meta.flavors.get(FLAVOR_NAME)
105    if conf is None:
106        raise MlflowException(
107            f'Model does not have the "{FLAVOR_NAME}" flavor',
108            RESOURCE_DOES_NOT_EXIST,
109        )
110    model_py_version = conf.get(PY_VERSION)
111    if not suppress_warnings:
112        _warn_potentially_incompatible_py_version_if_necessary(
113            model_py_version=model_py_version,
114        )
115    if CODE in conf and conf[CODE]:
116        code_path = os.path.join(local_path, conf[CODE])
117        mlflow.pyfunc.utils._add_code_to_system_path(code_path=code_path)
118    data_path = os.path.join(local_path, conf[DATA]) if (DATA in conf) else local_path
119    model_impl = importlib.import_module(conf[MAIN])._load_pyfunc(data_path)  ## type: ignore
120    return PyFuncModel(model_meta=model_meta, model_impl=model_impl)

 50        Generate model predictions.
 51        If the model contains signature, enforce the input schema first before calling the model
 52        implementation with the sanitized input. If the pyfunc model does not include model schema,
 53        the input is passed to the model implementation as is. See `Model Signature Enforcement
