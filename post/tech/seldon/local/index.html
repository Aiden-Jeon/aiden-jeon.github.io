seldon-core local에서 사용해보기

2021-04-22 · 2 min read  · seldon ·
Share on:

Overview

  • 1. Install
    • 1.1) seldon-core
    • 1.2) git clone
  • 2. Sklearn server
  • 3. MLflow server

이번 포스트에서는 seldon-core를 local에서 사용하는 법에 대해서 설명합니다.

1. Install

1.1) seldon-core

현재 seldon-core를 로컬에서 사용하기 위해서는 multiprocessing issue로 인해 3.7에서만 가능합니다.

1pyenv virtualenv 3.7.9 seldon
2pyenv activate seldon
3pip install seldon-core scikit-learn mlflow pandas

1.2) git clone

local에서 사용하기 위해서는 git repo를 clone해야 합니다.

1git clone https://github.com/SeldonIO/seldon-core

2. Sklearn server

clone받은 repo에서 sklearn server가 위치한 경로로 이동합니다.

1cd seldon-core/servers/sklearnserver/sklearnserver

간단한 iris를 분류하는 SVC 모델을 생성할 스크립트를 작성합니다.

 1# sklearn_model.py
 2import joblib
 3from sklearn import datasets
 4from sklearn import svm
 5
 6def train():
 7    df = datasets.load_iris()
 8    X, y = df.data, df.target
 9
10    clf = svm.SVC()
11    clf.fit(X, y)
12
13    joblib.dump(clf, "model/model.joblib")
14
15if __name__ == "__main__":
16    train()

모델 파일을 읽기 위해서는 경로가 필요합니다. 현재 위치를 경로로 설정할 수 없기 때문에 모델을 저장할 경로를 만듭니다.

1mkdir model

그리고 위에서 작성한 스크립트를 실행해 model.joblib을 생성합니다.

1python sklearn_model.py

이제 서버를 실행시켜 보겠습니다. SVC 모델은 predict 함수만 있기 때문에 parameter에서 predict method를 주겠습니다.

1seldon-core-microservice SKLearnServer --parameters '[{"name": "method", "type": "STRING", "value": "predict"}, {"name": "model_uri", "type":"STRING", "value": "file://model/"}]'

이제 predict를 해보겠습니다.

1curl -H "Content-Type: application/json" http://localhost:9000/predict -d '{
2    "data":{
3        "ndarray": [[0.6745011455,-0.5923730118, 1.0469454037, 1.1855672065]],
4        "names": ["sepal length (cm)", "sepal width (cm)","petal length (cm)","petal width (cm)"]
5    },
6    "meta" : {}
7}'

3. MLflow server

clone받은 repo에서 mlflow server가 위치한 경로로 이동합니다.

1cd seldon-core/servers/mlflowserver/mlflowserver

mlflow를 이용해 모델을 저장할 스크립트를 작성합니다.

 1# mlflow_model.py
 2import mlflow
 3from mlflow.models.signature import infer_signature
 4from sklearn import datasets
 5from sklearn import svm
 6
 7def train():
 8    df = datasets.load_iris()
 9    X, y = df.data, df.target
10
11    clf = svm.SVC()
12    clf.fit(X, y)
13    signature = infer_signature(X, clf.predict(X))
14    mlflow.sklearn.log_model(clf, "iris_model", signature=signature)
15
16if __name__ == "__main__":
17    train()

스크립트를 실행시키면 mlrus 폴더가 생성됩니다. 생성된 mlruns 폴더에서 artifact의 모델 경로를 찾습니다.

1cd mlruns/0/d96ad56315364063901ace2df62dfbc2/artifacts/iris_model

1> ls
2MLmodel    conda.yaml model.pkl

이제 서버를 실행시켜 보겠습니다. mlflow 서버는 predict 함수만을 지원합니다.

1seldon-core-microservice MLFlowServer --parameters '[{"name": "model_uri", "type":"STRING", "value": "file://mlruns/0/d96ad56315364063901ace2df62dfbc2/artifacts/iris_model/"}]'

이제 predict를 해보겠습니다.

1curl -H "Content-Type: application/json" http://localhost:9000/predict -d '{
2    "data":{
3        "ndarray": [[0.6745011455,-0.5923730118, 1.0469454037, 1.1855672065]],
4        "names": ["sepal length (cm)", "sepal width (cm)","petal length (cm)","petal width (cm)"]
5    },
6    "meta" : {}
7}'
